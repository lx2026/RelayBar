RelayBar keeps your SSH port forwards one click away in the macOS menu bar.

Paste a command you already use, such as:

ssh -N -L 3000:127.0.0.1:3000 user@host

RelayBar fills in the connection details, saves the tunnel locally, and gives you a clear start or stop control. It uses the SSH client built into macOS and shows useful connection failures directly in the menu.

Highlights:
• Native, compact menu-bar interface
• Quick import for common SSH local forwards
• Manual setup with only the essential fields
• One-click start and stop
• Clear running and error states
• No accounts, analytics, advertising, or cloud service
• Tunnel definitions remain on your Mac

RelayBar manages one local SSH forward per item and runs SSH non-interactively. Key or agent authentication is recommended.
